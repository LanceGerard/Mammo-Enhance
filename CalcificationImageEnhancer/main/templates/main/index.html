BMD Calcification Image Enhancer

A Web-based Image Enhancement on Microcalcifications Captured from Mammogram

This web application is purposely created to be able to fully and effectively enhance mammographic images for better detection and classification by applying select image enhancement techniques which includes intensity and contrast manipulation, filtering, equalization, segmentation, etc. The objective of this project is also to be beneficial for the industry and the academe. The results of this project may prove useful for doctors and specialists for an easier detection of microcalcification since it is an important indicator of early breast cancer. Students and researchers may also work with or upon the methodologies, findings and conclusions of this study to further the understanding within this subject matter.



This project was created through the collaborative effort of Hannah Mae Baroma, Jennylyn Basagre, Lance Gerard Daet, and Jessie Melendres
